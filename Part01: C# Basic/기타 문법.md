### Generic
class MyClass<T> { public T data; ... }와 같이 <T>를 이용해 다양한 type에 대응할 수 있게 구현하는 것을 의미. 함수에도 적용 가능하다.
#### where
  where은 Type T에 조건을 부여하는 키워드이다. Func<T>(T val) where T : class /struct / new() / MyClass ... 와 같이 사용한다.
